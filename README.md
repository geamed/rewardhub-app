# RewardHub (Next.js + React)

هذا المشروع الآن يستخدم Next.js بدلاً من Vite.

## تشغيل محليًا

**المتطلبات:** Node.js

1. تثبيت الاعتماديات:
   `npm install`
2. ضع متغير البيئة `GEMINI_API_KEY` في ملف [.env.local](.env.local)
3. لتشغيل التطبيق:
   `npm run dev`

## البناء والتشغيل للإنتاج

```bash
npm run build
npm start
```
